REQUIREMENTS:
	Blender v2.83+
	
Code had been tested on:
	Blender v2.83, v2.90,v2.91alpha
	
Installation:
	Download mCBF-2d-3d_marker-extraction.py eta: < 1min
Usage:
	Open mCBF-2d-3d_marker-extraction.py using Blender's text editor (can be found under scripting panel).
	Run the code within Blender. Execution time depends on the number of frames needed to export. For 1000 frames execution times were around 1 minute.
	The code will iterate through each frame and export as a .csv file 2d and 3d coordinates for the objects of interest.
